        "Переменные"
1) Примитивные Духи
Простые, но мощные сущности:

int — Дух Числа (хранит целые значения)
Изначала он равен 0, подобно пустому кошелю.
double — Дух Точности (дроби и вещественные числа)
float — Дух Мерцающей Точности (Менее точнее double)
Изначала они равны 0.0, подобно песку замест воды.
boolean — Дух Истины (правда или ложь)
Изначала он равен false, подобно неактивному свитку.
char — Дух Символа (одиночный знак или руна)
Изначала он null, подобно разорванному свитку.
String — Свиток Текста (последовательность символов)
Изначала равен null, ибо он нуждается в вместилище,
куда бы можно было разместить свиток.

2) Обряд Создания
Три шага для призыва переменной:
Объявление — "Явись, сосуд!"
Пример: int gold;
Инициализация — "Наполнись силой!"
Пример: gold = 50;
Использование — "Сверши магию!"
Пример: System.out.println(gold);

(Неочевидно: char поменяется если добавить int,
то он переменится)
